# 🎨 完全基于 TDesign 的 UI 升级

## ✨ 设计理念

完全参考 **腾讯 TDesign** 设计系统，实现企业级数据中台的专业界面。

### TDesign 核心特点
1. **极简主义** - 去除所有装饰性元素
2. **功能优先** - 突出内容和数据
3. **统一规范** - 严格的设计系统
4. **专业克制** - B端产品的专业感

## 🎯 主要改进

### 1. 配色方案 - TDesign 标准色

```scss
// 品牌色
主色: #0052d9 (TDesign Blue)
辅助色1: #029cd4
辅助色2: #0594fa
成功色: #00a870
警告色: #ed7b2f
错误色: #d54941

// 中性色
黑色-1: #000000e6 (90% 标题)
黑色-2: #000000d9 (85% 正文)
黑色-3: #00000073 (45% 次要文字)
灰色-1: #f3f3f3 (背景)
灰色-2: #e7e7e7 (边框)
灰色-3: #dcdcdc (分割线)
```

### 2. 圆角系统 - 统一 3px

```scss
// TDesign 使用极小的圆角
卡片: 3px
按钮: 3px
输入框: 3px
标签: 3px
```

### 3. 间距系统 - 4的倍数

```scss
超小: 4px
小: 8px
中: 12px
常规: 16px
大: 20px
超大: 24px
```

### 4. 边框系统

```scss
常规边框: 1px solid #dcdcdc
分割线: 1px solid #e7e7e7
表格边框: 1px solid #f3f3f3
```

### 5. 阴影系统 - 极度克制

```scss
// TDesign 几乎不使用阴影，只在 hover 时添加
hover 阴影: 0 1px 10px 0 rgba(0, 82, 217, 0.1)
```

## 🎨 设计对比

### 之前的设计问题
1. ❌ 过多的渐变色（紫色、粉色等）
2. ❌ 过大的圆角（10px、12px）
3. ❌ 过于花哨的装饰元素
4. ❌ 不统一的配色方案
5. ❌ 过度使用阴影效果

### 现在的 TDesign 风格
1. ✅ 统一的 TDesign 蓝色系
2. ✅ 极小的圆角（3px）
3. ✅ 去除所有装饰元素
4. ✅ 严格的配色规范
5. ✅ 几乎不使用阴影

## 📐 设计规范

### 字体规范
```scss
// 标题
h1: 20px / 600
h2: 18px / 600
h3: 16px / 600

// 正文
正文: 14px / 400
小字: 12px / 400

// 颜色
标题: #000000e6
正文: #000000d9
次要文字: #00000073
```

### 按钮规范
```scss
// 主按钮
背景: #0052d9
文字: #ffffff
圆角: 3px
高度: 32px (small: 24px, large: 40px)

// 文字按钮
文字: #0052d9
hover: #0034b5
```

### 表格规范
```scss
// 表头
背景: #f3f3f3
文字: #000000d9
字重: 500

// 表格行
边框: 1px solid #f3f3f3
hover: #f3f3f3
```

### 卡片规范
```scss
背景: #ffffff
边框: 1px solid #dcdcdc
圆角: 3px
内边距: 16px-24px
```

## 💡 TDesign 设计原则

### 1. Less is More
- 去除一切不必要的装饰
- 只保留功能性元素
- 让内容成为焦点

### 2. 一致性
- 统一的配色方案
- 统一的圆角大小
- 统一的间距系统

### 3. 专业性
- 克制的颜色使用
- 清晰的层次结构
- 适合B端产品

### 4. 功能优先
- 突出数据和内容
- 优化交互体验
- 提升可读性
