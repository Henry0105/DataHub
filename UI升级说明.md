# 🎨 UI 设计升级 - TDesign 风格

## ✨ 升级内容

参考腾讯 TDesign 设计规范，对整个系统进行了全面的 UI 升级！

### 设计理念
- 🎯 **简洁专业** - 去除过度装饰，突出内容
- 📐 **层次清晰** - 合理的间距和层级关系
- 🎨 **配色克制** - 使用中性色系，品牌色点缀
- 💼 **企业级** - 适合B端产品的专业感

## 🔄 主要改进

### 1. 登录页面 🔐
**之前**：紫色渐变背景（过于花哨）
**现在**：腾讯蓝渐变 + 几何装饰

#### 改进点：
- ✅ 使用 TDesign 品牌色 `#0052d9`
- ✅ 添加微妙的背景装饰元素
- ✅ 更大的输入框和按钮（提升可用性）
- ✅ 圆角从 10px 调整为 12px（更现代）
- ✅ 阴影更柔和自然

### 2. 主布局 📐
**之前**：深色侧边栏（#001529）
**现在**：白色侧边栏 + 浅灰背景

#### 改进点：
- ✅ 侧边栏改为白色，更清爽
- ✅ 菜单项圆角设计，hover 效果更友好
- ✅ 激活状态使用浅蓝色背景
- ✅ 更细腻的边框和阴影
- ✅ 图标和文字颜色统一

### 3. 首页看板 📊
**之前**：强烈的渐变色卡片
**现在**：简洁的白色卡片 + 微妙阴影

#### 改进点：
- ✅ 卡片使用白色背景
- ✅ 统计数字更大更清晰（32px）
- ✅ 模块卡片 hover 效果更自然
- ✅ 整体间距更合理（24px）
- ✅ 边框和阴影更克制

### 4. 全局样式 🌐
#### 改进点：
- ✅ 字体使用 PingFang SC（更适合中文）
- ✅ 背景色统一为 `#f3f4f6`
- ✅ 文字颜色使用 Tailwind CSS 色板
- ✅ 滚动条更细更优雅（6px）

## 🎨 配色方案

$primary: #0052d9;
$primary-hover: #0034b5;
$primary-light: #e6f0ff;
$primary-lighter: #f0f5ff;

// 中性色（Tailwind CSS）
$gray-50: #f9fafb;
$gray-100: #f3f4f6;
$gray-200: #e5e7eb;
$gray-400: #9ca3af;

## 📏 设计规范

### 间距系统
- 超小：4px
- 小：8px
- 中：16px
- 大：24px
- 超大：32px

### 圆角系统
- 小：4px（标签、徽章）
- 中：6px（按钮、输入框）
- 大：8px（卡片）
- 超大：12px（模态框）

### 阴影系统
```scss
// 轻微阴影
box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.03);

// 常规阴影
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);

// 强调阴影
box-shadow: 0 20px 60px rgba(0, 0, 0, 0.15);
```

### 字体系统
```scss
// 标题
h1: 32px / 600
h2: 28px / 600
h3: 24px / 600
h4: 20px / 600

// 正文
body: 14px / 400
small: 13px / 400
tiny: 12px / 400
```
