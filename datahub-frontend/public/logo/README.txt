组件Logo图标说明
=================

请将以下组件的Logo图标文件放置在此目录下，文件名格式为：组件名.png

需要的图标文件：
1. hadoop.png - Apache Hadoop
2. zookeeper.png - Apache Zookeeper  
3. kafka.png - Apache Kafka
4. flink.png - Apache Flink
5. spark.png - Apache Spark
6. kyuubi.png - Apache Kyuubi
7. seatunnel.png - Apache SeaTunnel
8. hive.png - Apache Hive
9. dolphinscheduler.png - Apache DolphinScheduler
10. minio.png - MinIO
11. filebeat.png - Filebeat
12. trino.png - Trino
13. amoro.png - Apache Amoro
14. dinky.png - Dinky

图标建议尺寸：48x48 像素或更大（会自动缩放）
图标格式：PNG（支持透明背景）

如果图标文件不存在，系统会自动显示彩色字母占位符。
